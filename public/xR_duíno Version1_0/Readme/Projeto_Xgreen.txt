Arduino feito com Uma caixinha bluethot, um pendrive e uma configuração de relé sônica...

-------------------

ideia base:

Em um pen drive, grava-se as sete notas musicais em suas devidas frequências, no formato mp3.

Ao reprozduzir as notas musicais na caixa de som o fone deverá ser removido, para que as ondas das frequências do pen drive reproduzida pela caixade som sejam usadas para manipular outros circuitos... 

Drives de relés sônicas devem ser configuradas para manipulações de aparelhos ou mesmo portas lógicas... etc...
-------------------

============

Considerando que a configuração de hardware funcione perfeitamente a ideia seguinte será programar a IDE de manipulação de dados para facilitar o manuseio de comandos...


Dentro da lógica de programação em blocos, A IDE deve fornecer um menu de audios e devidas frequências em formato mp3 para serem organizados numa area de trabalho... possibilitando tambem gravações de audio... ao organizar a sequencia de audios e frequencias desejada a mesma deve ser salva nO pendrive para ser executado no arduino"Xgreen"
